  • git commit -m "mais 8 li de comandos Git"
  • git reflog (histórico de atualizações)
  • git reset --hard b3e42ed (versão anterior)

  • git branch (ver quais estão disponíveis)
  • git branch stanging (criar uma nova branch com o nome de staging)
